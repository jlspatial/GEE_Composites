Examples include: area of England and New Zealand, with comprehensive coverage of both (check the time periods). 

New Zealand Imagery is a single-year composite of images from 2023. Code will automatically produce a true-colour and NDVI composite for visualisation in GEE.

England imagery is a three-year composite of images (2017-2019) in the spring-summer period.
